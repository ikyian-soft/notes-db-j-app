# Notes DB app

Simple test app for managing user notes.
The notes are kept in MySQL DB.

## Features

1. multi-user with authentication (users put in db with SQL scripts)
2. uses Bcypt password encryption
3. CRUD management of user records
4. Login/logout

## Configuration

1. create DB
2. Initialize DB with scripts from ´sql-scripts/´.
3. configure credentials for db access (default are for the local development).
